One of my users asks me, he wants export the SharePoint library folder structure.so he can put the structure in his presentation. While searching I saw the site (http://www.experts-exchange.com/articles/2802/Displaying-the-Windows-Directory-as-a-Visio-Organization-Chart.html) descript the way to draw the windows folder structure in Visio. Based on that idea I have finished the requirement as below.

Export data:

Using Export to Excel option library I have export the data.


  




Alter Data:
I have removed the unwanted data.


Then Change the column from path to Parent



Again I am creating one column named as Child with the following formula
=CONCATENATE(b2,"/",a2)


 Now my excel look's as below

And I have add the root element in the excel
Name: Shared Document
Parent: sites/audit/test/Shared Documents
Child: sites/audit/test/Shared Documents

Final Excel sheet looks like below,

Save & Close the excel

Import the data in Visio:

In Visio Choose the organization Chart template and create



Choose Org Chart & Click import



Choose first option



Choose Excel file as option


  
Browse the Excel and click next



In Next window Choose Name as Child and Reports to as parent



Choose required fields in next two windows







On Next window Choose auto page break and choose share document as top page



Click finish

You will get the folder structure.
