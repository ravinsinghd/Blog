Now we are see about Notification Tool in vb.net.That use to manage your tray icons in windows os

Step1:

Open a new form

Add the Notify icon tool to your form

In property pane just set any icon file for icon property

Then Just create two buttons in form as follow

Public Class Form1

    Private Sub Button1_Click(sender As System.Object, e As System.EventArgs) Handles Button1.Click

        NotifyIcon1.BalloonTipText = "hi"
        NotifyIcon1.ShowBalloonTip(1)


   End Sub

    Private Sub Button2_Click(sender As System.Object, e As System.EventArgs) Handles Button2.Click


       NotifyIcon1.BalloonTipText = "Hello"
        NotifyIcon1.ShowBalloonTip(1)


    End Sub
End Class

Now run the form click button and see output
