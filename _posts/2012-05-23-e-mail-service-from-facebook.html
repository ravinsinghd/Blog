Hi friends!!!!

Today we are see about how we use Facebook as mail server.The facebook add the feature that can be enable we use the facebook as mail client.

Step1:

Go to facebook account Setting

In general tab click Edit in username

If you alreaty set the username then the username will display or else set new username

The username is your email address

For example my username is ravin.singh75 so that my mail address is ravin.singh75@facebook .com

Now you send email for this address

You can send mail with subject and also attachments

The received mail are display in your message tab in facebook
