Hi friends!!!!
Today we are see about datagridview control in vb.net
The tool will be use to view your database completely in vb form completely
Step 1
Open New Form
Add Data Grid view control to the form

step 2
Create one access database named as database1
In database1 create new table named as usertablelist

Note:
if you change the database or table name then please update the code as respectively
Step 3:
Then import the oledb package
Initialize the oledb data tools
Then form load event write the code
The above three steps code are as follows
Imports System.Data.OleDb
Public Class Form1
    Dim my_dbConnection As New OleDbConnection("Provider=Microsoft.ACE.OLEDB.12.0;Data Source=C:\Users\ravin singh\Desktop\college\Database1.accdb")
    Dim my_Command As New OleDbCommand
    Dim my_adapter As New OleDbDataAdapter
    Dim my_dataset As New DataSet
 
Private Sub Form1_Load(sender As System.Object, e As System.EventArgs) Handles MyBase.Load
        my_adapter = New OleDbDataAdapter("select * from usertablelist", my_dbConnection)
        my_adapter.Fill(my_dataset)
        Me.DataGridView1.DataSource = my_dataset.Tables(0)
    End Sub
End Class
Note:
Please change the connection path as your database path
Step 4:
Now Run the Form Datagridview display your data
