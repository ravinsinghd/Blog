Today We are See about Pop the Forms from Task bar Using VB.Net

STEP 1:
  1. Open new project
  2. Open New Form Names as Form1
  3. Add Following Control to the form          
  •            2 Timers
  •            1 Button
STEP 2:
  1. Set the form Border Style as None using properties  
  2. Set the form size As (223, 201) using Properties 
Step 3: 
  1. Define variable i ,a,b as integer
  2. In form Load event Just write the following code

a= Screen.PrimaryScreen.Bounds.Width
        b = Screen.PrimaryScreen.Bounds.Height
        a= a - 227
        i = 25
        Timer1.Start()

     3.Then in Button 1 click event write following code

 Timer2.Start()

      4.Then in Timer 1 Tick Event Write The following Code

 If i > 31 Then
            Timer1.Stop()
        End If
        b = b- i
        Me.Location = New System.Drawing.Point(a, b)
        i = i + 1

      5.Then in timer 2 Tick Event Write the following code

 If i < 25 Then
            Timer2.Stop()
            Me.Close()
        End If
        b= b + i
        Me.Location = New System.Drawing.Point(a, b)
        i = i - 1

Now Run Form The form will appear popup from task bar
